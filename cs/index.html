UI Components App

Použité technologie:

  • 🟢 HTML, CSS – struktura a základní stylování aplikace
  • 🟢 JavaScript (ES6+) – komponentová architektura, správa událostí a stavů
  • 🟢 LocalStorage – uchování dat (např. článků) mezi relacemi

Funkcionality:

  • ✅ Burger menu – zobrazí navigaci na menších obrazovkách, reaguje na klik mimo komponentu
  • ✅ Accordion – skládací seznam s dynamickým obsahem z JS objektů
  • ✅ Blog – přidávání článků s titulem, obsahem a autorem, data ukládána do localStorage
  • ✅ Vyhledávání – fulltextový filtr článků podle názvu nebo autora, se zobrazením „No results found“
  • ✅ Pagination – stránkování blogových článků podle jejich počtu a aktuálního filtru
  • ✅ Komponentová struktura – každá funkce (accordion, pagination atd.) jako samostatný JS modul

Tato aplikace slouží jako tréninková platforma pro budování znovupoužitelných UI komponent. Každá komponenta je izolovaná, snadno spravovatelná a dynamicky reaguje na uživatelské akce. Projekt je navržen s důrazem na čistý kód, škálovatelnost a praktické využití v budoucích aplikacích.

Habit List App

Použité technologie:

  • 🟢 HTML, CSS – struktura a základní stylování aplikace
  • 🟢 JavaScript (ES6) – logika sledování návyků a práce s localStorage

Funkce:

  • ✅ Přidání návyku – zadáním názvu a popisu vytvoříš nový návyk
  • ✅ Sledování pokroku – každý den můžeš označit, že je návyk splněn
  • ✅ Smazání návyku – jednotlivé návyky lze odstranit kliknutím na ikonu koše
  • ✅ Uložení dat – všechny návyky a jejich pokrok se ukládají do localStorage
  • ✅ Přehledná a modulární struktura – aplikace využívá znovupoužitelné komponenty a pomocné funkce

Tato aplikace pomáhá uživatelům budovat dobré návyky tím, že umožňuje sledovat jejich každodenní plnění. Každý návyk je zobrazen formou karty s názvem, popisem a přehledem pokroku. Data se ukládají lokálně, takže se po obnovení stránky nic neztratí.
