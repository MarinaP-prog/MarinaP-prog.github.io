Habit List App

Technologies Used:

  • 🟢 HTML, CSS – structure and basic styling of the app
  • 🟢 JavaScript (ES6) – habit tracking logic and localStorage integration

Features:

  • ✅ Add a habit – enter a title and description to create a new habit
  • ✅ Mark progress – each day, users can check off habits to track consistency
  • ✅ Delete a habit – habits can be removed individually using the delete icon
  • ✅ Persistent data – all habits and their progress are saved in localStorage
  • ✅ Clean and modular code – the app uses reusable components and utility functions

This application helps users build good routines by allowing them to track daily habits. Each habit is represented by a card that displays its title, description, and progress. Data is stored locally so that users can continue where they left off even after refreshing the page.
